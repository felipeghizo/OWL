----- OPERADORES:

	----- OPERADORES ARITIMÉTICOS:

		+	->	Soma.
		-	->	Subtração.
		*	->	Multiplicação.
		/	-> 	Divisão
		%	->	Módulo (resto da divisão).
		**	-> 	Elevado.

		n +=

		 */

	----- OPERADORES RELACIONAIS:
	
		>	->	Maior que.
		<	->	Menor que.
		>=	->	Maior ou igual.
		<=	->	Menor ou igual.
		==	->	Igual.
		===	->	Identico.
		!=	->	Diferente.


	----- OPERADORES LÓGICOS:

		!	->	Negação (NOT).
		&&	->	Conjunção (AND).
		||	->	Disjunção (OR).

	----- OPERADOR TRENÁRIO:
		
		// Usado em estruturas de condições
		teste ? True : False

	----- PRECEDENCIA:
		() -> ** -> (*, /, %) -> (+, -) -> (>, <, >=, <=, ==, ===, !=) -> ! -> && -> ||
